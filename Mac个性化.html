JEnv

借助JEnv快速管理和切换本机的Java版本。

安装

brew install jenv

配置

bash

zsh

使用

  • /usr/libexec/java_home -V 命令罗列出已经安装的 JDK 路径
  • jenv add 路径 添加路径
  • jenv versions 查看已经被添加至 JEnv 管理的 JDK 版本
  • jenv shell/local/global oracle64-1.8.0.271 shell 用于设置终端窗口生命周期内使用的 JDK 版本；local 用于设置当前目录下使用的 JDK 版本；而 global 用于设置全局使用的 JDK 版本
