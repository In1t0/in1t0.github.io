抓住那只猫

打开页面，天真的我还以为这个是命令执行……

?url=127.0.0.1

web题菜鸡新思路:打开是php页面，后端不一定是php语言，还可能是python！

特殊字符外加宽字节fuzz一下 #python的题一般都要fuzz一下特殊字符#

?url=%df发现报错信息。注意直接在地址栏里fuzz，在网页上的输入框上fuzz浏览器会进行二次url编码。


当 CURLOPT_SAFE_UPLOAD 为 true 时，PHP 可以通过在参数中注入 @ 来读取文件。当且仅当文件中存在中文字符的时候，Django 才会报错导致获取文件内容。


通过 Django 报错调用栈中的信息，请求 @/opt/api/api/settings.py 得到数据库名称，在通过 @/opt/api/database.sqlite3 得到数据库内容，其中包含 Flag。
